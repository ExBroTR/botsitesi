All Commands

.help - Shows the help categories.
.pp/.avatar/.av - Shows the profile picture of a member.
.info - Shows a member’s information.
.afk (status) - Set an AFK status to display when you are mentioned.
.server-info - Shows the server’s information.
.invite - Invite and server links of the bot.
.link - Invite link of the Bot’s official server.
.spotify - Shows the information of the song that the person is listening to at the moment.
.snipe - Snipes the last deleted message in the channel.
.discrim - Shows a list of the users with the same tag (ern#2088) as you.

.ban [user] (reason) - Bans a member from the server.
.baninfo [ID] - Shows the reason for a banned user.
.unban [ID] - Unbans a user.
.kick [user] (reason)
.mute [user] (time) (reason) - Mutes a member from all text channels.
.clear [1-100] - Clears messages.
.profanity (enable/disable) - Enables/Disables profanity filter.
.advertisement (enable/disable) - Enables/Disables self-advertisement blocker.
.lock/.unlock - Locks/Unlocks the typed or tagged channel.
.autorole help - Type to view Auto-Role commands.
.jail-help - Type to view Jail commands.
.poll - Starts a poll in the typed text channel.
.giverole/.takerole [user] [role] - Gives/Removes a specified role from the tagged member.
.roleinfo [role] - Shows the info of a role.
.language [tr/en] - Changes the language of the bot.
.prefix set/reset - Changes the prefix.
.nuke - Purge all messages in a channel.
.setnick [user] (new name) - Changes the nickname of the tagged member.
.boost - Assigns the people who boost the server to the channel you specify.
.forbidden-word help - Type to view the Forbidden Words command.
